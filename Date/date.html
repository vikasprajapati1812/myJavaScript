Date

        constructor:    The constructor property returns the function that created the Date prototype
                        var d= new Date();
                        let text=d.constructor;
        Date.parse:  parse() parses a date string and returns the time difference since January 1, 1970.
                     parse() returns the time difference in milliseconds.
      
        getDate()           The getDate() method returns the day of the month (1 to 31) of a date.
      
        getDay()        method returns the day of the week (0 to 6) of a date.
                        Sunday = 0, Monday = 1, ... (See below):
        getFullYear()        return the year of the specific date "YYYY"
      
        getHours() ====>     get the hours from the specific date

        
        getMinutes()===>    get the minutes of the specific date.

        getSeconds()===>    returns the seconds of the specific date.
        
        getMilliseconds() returns the milliseconds (0 to 999) of a specific date.

        getMonth()====>     return the months in no.   Jan=0 feb=1 ..... dec=11
        
        getTime()           returns the number of milliseconds since January 1, 1970 00:00:00.

        Date.now()           returns the number of milliseconds since January 1, 1970.
                        const minute = 1000 * 60;
                        const hour = minute * 60;
                        const day = hour * 24;
                        const year = day * 365;
                        let years = Math.round(Date.now() / year);
