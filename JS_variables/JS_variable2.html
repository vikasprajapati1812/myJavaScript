Variables

  1. 1. A variable is a label that references a value.
  2. 2. Use the let keyword to declare a variable.
  3. 3. An undefined variable is a variable that has been declared but not initialized while an undeclared variable is a variable that has not been declared.
  4. 4. Use the const keyword to define a readonly reference to a value.

Var

variables can be declared in 3ways, var let and const

var are hoisted and are in global scope. var are declare initialise at any time and can be redeclared.

let

let and const are also said to be hoisted but they go through the temporal dead zone (tdz).

let cannot redeclared and it can be initialised later .it has a block scope

const

Const cannot redeclared and it cannot be initialised later . it is declared and initialised at the same time. it has a block scope
